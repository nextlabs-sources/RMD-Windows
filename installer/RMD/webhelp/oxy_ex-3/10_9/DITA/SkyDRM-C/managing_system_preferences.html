Managing system preferences

You can set system preferences, which are the default behavior of the SkyDRM Desktop for Windows app in your computer.
  1. From the menu bar, select Preferences > System.
  2. Define your system preferences by selecting any of the following radio buttons.
    • Show desktop notifications.

      Selecting this option enables SkyDRM to show SkyDRM activities in your desktop notification area.

    • Keep an offline copy when the protected file (.nxl) is saved to SkyDRM.
  3. Click Apply if you want to continue making changes to other preferences.
  4. Click Save to apply your preferences and close the dialog box.
